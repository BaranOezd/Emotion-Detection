Welcome to Emotion Analyzer!

1 Adding Text

You can type directly in the text editor or use the "Upload" button to analyze a text file.
Good Practice: Don't forget about the punctiations!

2 Understanding Line Chart

The left panel shows emotion trends across your text. Each line represents an emotion. Click checkboxes to filter emotions, or use Shift+Click for multiple selections.

3 Using Bar Chart

The right panel shows emotion levels for the selected sentence. Drag the bars to adjust emotion intensity. Use the Undo button to reset changes to their original values.

4 Rewriting Sentences

After adjusting emotions, click "Rewrite" to generate a new sentence. If you don't like the changes, use "Undo" to previous the original sentence and emotion levels.
